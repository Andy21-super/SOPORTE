# Manual de usuario

## Inicio de sesion

Ingresar con usuario y contrasena. Usuario inicial:

- `CD.ADMIN`
- Ingresar la clave asignada al administrador

## Dashboard

Muestra tickets abiertos, cerrados, criticos, reabiertos, vencidos, SLA y usuarios activos.

## Crear ticket

Seleccionar modulo, categoria, prioridad, asunto y descripcion. El sistema genera automaticamente el numero del ticket, fecha, usuario, area, cargo, estado inicial y SLA.

## Seguimiento

Desde el detalle del ticket se visualiza la conversacion, estado, prioridad y trazabilidad. Al responder con `No solucionado`, el sistema reabre el ticket.

## Reportes

El panel de reportes permite exportar tickets a CSV. La estructura queda preparada para Excel y PDF.

## Configuracion

El administrador puede gestionar parametros como SMTP, SLA, colores, logo, categorias, modulos y roles desde el modulo de configuracion.
